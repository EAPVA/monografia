\chapter{An\'{a}lise de Riscos}\label{cap:analiseRisco}

\noindent{\textbf{Dano no \emph{kit} de desenvolvimento}}

O \emph{kit} de desenvolvimento pode ser danificado, inviabilizando o projeto.

\textbf{Impacto:} 0 - Crítico

\textbf{Probabilidade:} 1 - Média 

\noindent \textbf{Mitigação:}
\begin{itemize} 
  \item Adquirir \emph{kits} adicionais.
\end{itemize}

\textbf{Impacto atualizado:} 0 - Crítico

\textbf{Probabilidade atualizada:} 2 - Baixa

\vspace{7mm}

\noindent{\textbf{Desenvolvimento em Cuda complexo demais}}

O desenvolvimento de código utilizando Cuda pode ser complexo demais,
dificultando a realização do projeto no tempo previsto

\textbf{Impacto:} 0 - Crítico

\textbf{Probabilidade:} 2 - Baixa

\noindent \textbf{Mitigação:}
\begin{itemize} 
  \item Priorizar alocação de horas-homem para esta atividade.
  \item Alocar atividades relacionadas a Cuda para começarem o mais cedo
  possível.
  \item Pesquisar alternativas para trabalhar com processamento paralelo.
\end{itemize}

\textbf{Impacto atualizado:} 1 - Significativo

\textbf{Probabilidade atualizada:} 2 - Baixa

\vspace{7mm}

\noindent{\textbf{Atividades não relacionadas ao projeto ou imprevistos tomarem
tempo excessivo da equipe}}

Como os integrantes da equipe não trabalham em dedicação exclusiva ao projeto,
tendo outras atividades acadêmicas e profissionais, estas podem prejudicar o
tempo disponível ao projeto.

\textbf{Impacto:} 1 - Significativo

\textbf{Probabilidade:} 1 - Média

\noindent \textbf{Mitigação:}
\begin{itemize} 
  \item Realizar atividades com a maior antecedência possível.
  \item Priorizar este projeto em relação a outras atividades acadêmicas.
\end{itemize}

\textbf{Impacto atualizado:} 0 - Baixo

\textbf{Probabilidade atualizada:} 1 - Média

\vspace{7mm}
\noindent{\textbf{Um dos integrantes deixar a equipe}}

Um dos integrantes, por motivos de: dificuldades com horário, desentendimento
com a equipe, doença ou falecimento, pode vir a deixar a equipe.

\textbf{Impacto:} 1 - Significativo

\textbf{Probabilidade:} 2 - Baixa

\noindent \textbf{Mitigação:}
\begin{itemize} 
  \item Permitir horários flexíveis para que nenhum integrante tenha que sair
  por problema de tempo.
  \item Comunicação e reuniões frequentes.
  \item Manter um ambiente de trabalho harmônico.
  \item Evitar a concentração de conhecimento em um dos membros da equipe.
\end{itemize}

\textbf{Impacto atualizado:} 2 - Baixo

\textbf{Probabilidade atualizada:} 2 - Baixa

\vspace{7mm}

\noindent{\textbf{O \emph{kit} de desenvolvimento não ser rápido o suficiente
  para rodar em tempo real}}

O \emph{kit} de desenvolvimento pode não possuir poder de processamento
suficiente para executar o algoritmo em tempo real (um \emph{frame} a cada
50ms). Sendo um projeto de pesquisa, se considerou que o impacto dessa falha é
baixo.

\textbf{Impacto:} 2 - Baixo

\textbf{Probabilidade:} 1 - Média 

\noindent \textbf{Mitigação:}
\begin{itemize} 
  \item Nenhuma estratégia de mitigação conhecida. 
\end{itemize}

\textbf{Impacto atualizado:} 2 - Baixo

\textbf{Probabilidade atualizada:} 1 - Média 
\vspace{7mm}

\noindent{\textbf{Treinamento do \emph{T-HOG} gerar baixa taxa de acertos}}

O algoritmo proposto pode não apresentar uma performance suficiente. Sendo um
projeto de pesquisa, se considerou que o impacto dessa falha é baixo.

\textbf{Impacto:} 2 - Baixo

\textbf{Probabilidade:} 2 - Baixa

\noindent \textbf{Mitigação:}
\begin{itemize} 
  \item Alocar um tempo razoável para fazer o \emph{tuning} do algoritmo 
\end{itemize}

\textbf{Impacto atualizado:} 2 - Baixo

\textbf{Probabilidade atualizada:} 2 - Baixa
