\chapter{Conclusão}\label{cap:conclusao}

\begin{enumerate}
\item Localização das placas é um grande consumidor de tempo de
	processamento em sistemas de reconhecimento de veículos;
\item Sistemas de reconhecimento placas são tecnicamente viáveis;
\item já são aplicadas em outros países;
\item a tecnologia tem potencial para causar redução de custos ao substituir
	outros sistemas;
\item existe interesse governamental para fins de planejamento e fiscalização;
\item Sistemas modernos de computação móvel possuem GPU
\end{enumerate}

Com base nestes fatos acreditamos que:
com software suficientemente otimizado
é possível produzir software de reconhecmento de veículos usando plataforma
de baixo custo;
a etapa de localização das placas na imagem é um excelente
candidato a otimização via delegação para GPU;
o tema de localização de placas em uma stream de vídeo seja pertinente
academicamente e realizável, data a capacidade financeira e disponibilidade
de tempo dos autores deste trabalho.
