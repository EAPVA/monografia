\section{Recursos}


Para o desenvolvimento do projeto os seguintes recursos serão usados:

\begin{itemize}
\item 3 x \emph{Kit} de desenvolvimento Jetson TK1
\item 3 x \sigla{PC}{Personal Computer}s de desenvolvimento
\item 1 x Monitor/\sigla{TV}{Televisão} com entrada \sigla{HDMI}{High 
		Definition Multimedia Interface}(\emph{High Definition
		Multimedia Interface}) para uso com o \emph{kit} de
		desenvolvimento
\item 3 x \sigla{SDK}{Software Development Kit} Cuda - Cuda\emph{Toolkit}
\item Documentação do kit de desenvolvimento disponível no \emph{site} da 
		NVIDIA
\item 3 x Sistema Operacional Linux para os \emph{kits} de desenvolvimento
\item 3 x Engenheiro de Computação
\end{itemize}

\section{Viabilidade}

Para os custos de desenvolvimento, foram considerados apenas os gastos com mão 
de obra e materiais. Para os gastos com mão-de-obra foi tomado como referência 
a base do salário do \sigla{CREA}{Conselho Regional de Engenharia e Agronomia},
(Conselho Regional de Engenharia e Agronomia) oito salários mínimos, 
equivalente a R\$35,82/hora (base 2015). O valor escolhido foi
R\$40,00/hora.

Para os gastos materiais, foram estimados os valores de frete e imposto, e 
considerado o valor do câmbio no dia 27/11/2014. Foi também considerado que 
podem haver gastos em outros materiais, como cabos, cartões SD ou outros 
periféricos e componentes diversos, mas foi estimado que esse custo não será 
superior a R\$300,00.

Os PCs de desenvolvimento e o monitor com HDMI serão recursos externos doados 
ao projeto pelos integrantes da equipe, não entrando nos custos de projeto.
